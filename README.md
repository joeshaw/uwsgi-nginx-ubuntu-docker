# uwsgi-nginx-ubuntu

Based on [uwsgi-nginx](https://github.com/tiangolo/uwsgi-nginx-docker)
by [Sebastián Ramírez (@tiangolo)](https://github.com/tiangolo), but
using Ubuntu 12.04 as the base container rather than the official
Python container.
